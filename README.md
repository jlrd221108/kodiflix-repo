# Kodiflix Repository

Kodi repository for Kodiflix.

## File Manager Source

Add this source in Kodi File Manager:

```text
https://jlrd221108.github.io/kodiflix-repo/
```

Kodi should show:

```text
repository.kodiflix-1.0.0.zip
```

Install that zip first. Then install `Kodiflix`. Then, `Kodiflix Helper` will be installed automatically.
