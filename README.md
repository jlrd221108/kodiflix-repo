# Kodiflix Repository

This is the Kodi repository for Kodiflix.

## GitHub Pages URL

```text
https://jlrd221108.github.io/kodiflix-repo/
```

## What to upload

Upload the contents of this folder to a GitHub repository named `kodiflix-repo`. Keep the folder structure exactly as-is.

## Enable GitHub Pages

1. Open the GitHub repository.
2. Go to Settings > Pages.
3. Set Source to `Deploy from a branch`.
4. Set Branch to `main` and Folder to `/ root`.
5. Save.

## Kodi install zip

After GitHub Pages is live, install this zip in Kodi:

```text
zips/repository.kodiflix/repository.kodiflix-1.0.0.zip
```

Then install `Kodiflix Helper` first, then `Kodiflix`.
